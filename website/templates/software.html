My personal website

This website is built with pure HTML, CSS, JavaScript, and Python. Rather than using modern frameworks like Node.js, Vue.js, or Next.js, I chose a minimalist approach to maintain full control and flexibility in my codebase.


The website operates using Python (Flask) to handle routes through blueprints for the various web pages. User information (email and full name) submitted on the login/signup page is securely stored in a connected database using an API key. While I call it a login page, users simply input their email and full name, which redirects y'all to the homepage. In practice, this function is more like a signup form.


You can explore all the code on my GitHub repository. Feel free to clone it and tailor the project to suit your needs. While you'll have full access to the code, note that sensitive data such as my database connection keys and API keys are not included for security reasons.

My thesis project

My thesis defend last term was about computer vision and real-time color detection. THe main purpose of this is to learn about the pymodbus communication method between python and PLC.


But I wanna push my boundaries further so I create the whole "GUI - graphical user interface" that that allows users to interact with the machine and control it. I integrated in the GUI multiple buttons, lighting systems for warning the users if there could be sth wrong, and input labels, etc. The whole software was written in tkinter and customtkinter. The code will be also available on my github repo, fell free to clone it for your own usage.
