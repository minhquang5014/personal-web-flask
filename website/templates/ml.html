Convolutional Neural Network

Image classification

A CNN, or Convolutional Neural Network, is a type of artificial intelligence. It is mainly used for analyzing images. It works by breaking the image into small pieces and looking for patterns. These patterns help it recognize things in the image, like edges or shapes. CNNs are used in many applications, like facial recognition, self-driving cars, and medical image analysis. Some of the projects I have made included facial detection and recognition, object detection and object tracking, and image segmentation.

Long short term memory (LSTM)

A Recurrent Neural Network (RNN) is a type of neural network designed for processing sequences of data, such as time series, language, or video frames. Unlike traditional feedforward neural networks, RNNs have a unique architecture that includes loops, allowing them to maintain a hidden state that captures information about previous elements in the sequence. This makes them well-suited for tasks where the order of inputs matters, such as natural language processing (NLP) or sequence prediction. I've been working on car price prediction and it was a pretty fun project to build. A lot of new stuffs to learn, from preparing and processing the data to training the model and making the prediction.

GANs

Generative Adversarial Networks (GANs) are a class of machine learning models used for generating new data samples that are similar to a given dataset. GANs consist of two main components, a generator and a discriminator, which are trained simultaneously in a game-like scenario. I've been trying to generate cat and dog images based on a large number of datasets but it still has got some errors so I'll keep updating on that

Stable diffusion and large language model

Stable Diffusion Models are a type of generative model used for generating high-quality images, often from text prompts, based on a process called diffusion. These models are part of the broader category of Diffusion Models, which have recently gained attention for their ability to generate detailed and realistic images.
